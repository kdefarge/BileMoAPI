BileMoAPI is a Symfony API application with [api-platform](https://github.com/api-platform/api-platform)

## Installation

*   install BileMoAPI with composer

```bash
git clone git@github.com:kdefarge/BileMoAPI.git
cd BileMoAPI
composer install
```
